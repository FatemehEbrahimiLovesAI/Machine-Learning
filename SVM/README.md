# Vector Operations Visualization with Python


## Description
  In this project, I plan to implement everything mathematical I learn in the vectors section in preparation for entering SVM here.


## Requirements
  - Python > 3.8
  - Numpy


## Results

### Norms
  - v1_n : v1's Vector Norm
  - v2_n : v2's Vector Norm
    
### Dot Product
  - S_DP : Normal Dot Product
  - C_DP : Cos Dot Product
    
### Projection
  - v3_n : Projection Vector's Norm
    
### Vector Directions
  - v1_VD : for Vector 1
  - v2_VD : for Vector 2
